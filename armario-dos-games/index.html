Hello, World!


Seja bem vindo à minha página. Meu nome é Marcos Roberto. Este projeto começou quando iniciei meus estudos em HTML5 e CSS3. Além de ser um projeto onde aplico e treino conceitos, é também um jeito bacaninha de catalogar minha coleção de games.

Para começar clique no botão ao lado ou navegue pelo menu acima para ver os games organizados por console.

Aqui você vai ver algumas informações dos jogos como desenvolvedor e data de lançamento, quando comecei a jogar e em que pé está a jogatina: finalizei, abandonei, plantinei ou nem comecei 😁

Conforme vou avançando nos estudos, irei atribuir mais funcionalidades para o projeto e, se houver viabilidade, disponibilizar para comunidade como um serviço open source.
